# Software List for the Course

In order to engage fully with the course and achieve all of the goals, you'll need to set up the following software tools. These are all free to use and should be relatively easy to set up. If you encounter problems, don't hesitate to ask questions using GitHub issues.

---

## ![GitHub Logo](https://github.githubassets.com/images/modules/logos_page/GitHub_Account){width=28px} GitHub Account

[GitHub](https://github.com) is a developer platform that allows developers to host their code. This is also the platform we will be using to share the course materials. This can be a great place to see what other data scientists are working on.

### Setup Instructions:
1. Go to [GitHub](https://github.com) and click "Sign Up" (top right)
2. Enter:
   - Your email address
   - A password
   - A username

[Create your account now](https://github.com/signup){.button}
